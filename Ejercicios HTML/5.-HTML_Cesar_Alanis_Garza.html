El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.


Un elemento de HTML es un tipo de componente de documento de HTML, uno de los diversos tipos de nodos de HTML.


Una URL relativa es una versión abreviada de una URL absoluta. Su objetivo es eliminar todas las partes de la URL absoluta que se pueden adivinar a partir de la información de contexto de la página web.

Los marcadores delimitan elementos de un documento como cabeceras, párrafos, etc.


input> se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario.


label> representa una etiqueta para un elemento en una interfaz de usuario.
